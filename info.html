The data used in this web app is from the CORGIS Dataset Project and can be found here! The data contains information about recorded UFO sightings and their location, time frame, and shape. Because the data doesn't offer statistics on which state has the highest number of sightings, that information will not be presented on this website. The most common UFO shapes reported are orb, triangle,other, fireball, disc, egg, cigar, changing, rectangle and diamond.
